# PostUploader

为了极度犯懒产生的插件，构造请求，起飞！

功能如图，应该是一目了然了吧

- 切换颜色
- 控制请求头
- 控制请求体
- 制造一个1bit文件上传（回去自己抓包改啊！，懒狗！！）

chrome插件，符合v3使用，下载打开开发者模式（我知道你一定开了！），加载未打包的扩展程序
